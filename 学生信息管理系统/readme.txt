(1)第一次进入系统时提示管理员注册，再次进入时显示登录/注册界面
(2)只能注册普通用户，可由管理员提升权限，登录后可识别管理员/普通用户权限
(3)学生基本信息包括：班级、学号、姓名、三门课程（高等数学、大学英语和C语言）的成绩、考试平均成绩、考试名次、同学互评分、辅导员评分、班主任老师评分、综合测评总分、综合测评名次。考试平均成绩、同学互评分、辅导员评分、班主任老师评分分别占综合测评总分的70%、10%、10%、10%。提示：综合测评总分＝考试平均成绩*0.7+同学互评分*0.1+辅导员评分*0.1+班主任老师评分*0.1。
(4)管理员可提升普通用户权限
(5)能够输入学生基本信息，要求学号唯一，并按学号升序存入文件中。（需管理员权限）
(6)能够进行查找并显示相关信息——可按学号、姓名、班级进行查找，或进行模糊查找，按学号或姓名进行模糊查找。
(7)能够插入、删除、修改学生信息，并将修改后的信息再次写入文件。（需管理员权限）
(8)能够报表打印，导出为AllStudent文件或AllUsers文件，以csv格式存储，可直接在excel表格中打开。
